Project Title: Hand Made Tea Production Management System
Overview:
This project is a web-based application designed to digitize and streamline the manual data recording process of a specialized hand-made tea factory. The primary focus is to replace traditional paper-based logging with an efficient digital entry system that automates cost calculations and provides real-time data visualization.

Key Features:

Data Entry & Management: Easy-to-use interfaces for recording daily green leaf collection, tea types (Purple, Pink, Golden Tips, etc.), and dehydrator machine cycles.

Automated Cost Calculation: Automatically calculates the Cost of Production (COP) by integrating Green Leaf (G/L) costs, labor costs (selection and hand rolling), and electricity consumption.

Production Summaries: Generates monthly production summaries and inventory tracking for different tea varieties.

Resource Tracking: Monitors electricity usage (meter readings for dryers/rollers) and labor requirements for specific high-value tea types.

Sales & Inventory Insights: Tracks selling details across different channels (Tea Centre, Brokers) with support for multiple currencies (USD/LKR).

Tech Stack:

Frontend: React.js / HTML / CSS (Based on previous project architecture)

Backend: Node.js / Express.js

Database: MongoDB

Tools: Visual Studio Code, Git, Figma (UI Design)
